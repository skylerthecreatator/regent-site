Как тестировать

Обычный сценарий занимает пару минут.

1 Установи Setup Закрой старую версию Regent Connect перед установкой. Если установщик просит заменить файлы, соглашайся.
2 Нажми Smart В приложении выбери Smart Discord + YouTube. Для Telegram отдельно нажми Smart Telegram.
3 Проверь сервисы Открой Discord, YouTube и Telegram Desktop. Для Telegram включи MTProto proxy: 127.0.0.1, порт 1443.
4 Если не работает Открой Настройки -> Сформировать отчет. Отправь отчет, провайдера, город, Windows и короткое описание проблемы.

Что важно

Нужны не просто “работает/не работает”, а данные по провайдерам.

ПровайдерНапример: Дом.ру, Ростелеком, МТС, Билайн.
РегионГород или область. У разных регионов фильтрация может отличаться.
ПоведениеDiscord stuck на update, YouTube не грузит видео, Telegram не подключается к proxy и так далее.

Предупреждение Windows

Если браузер или SmartScreen пишет, что файл может быть опасен.

Почему так происходит У beta-приложения еще нет платной подписи издателя и репутации скачиваний. Windows часто предупреждает о новых `.exe`, даже если файл скачан с GitHub.
Как проверять Скачивай файл только с GitHub Releases. Размер setup около 3 МБ. Если файл получен из другого источника, лучше не запускать.
